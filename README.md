StopWatch App

The app includes the following features:
- Start / Stop timer
- Pause timer
- Reset timer
- Lap 
- Clear timer

Features:
The Start/Stop - toggles between starting and stopping the timer
Pause - Pauses the timer without resetting the time
Reset - Resets the timer to 0 and clears all laps
Lap - Records lap times and displays them in a list
Lap - You can tap a lap to delete it
